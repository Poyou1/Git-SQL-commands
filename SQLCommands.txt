Войти в настройки:

sudo mysql -u root -p
Docker:
mysql -h mysql -u root -p
Все базы данных на sql server:

SHOW DATABASES; 
Удалить базу:

DROP DATABASE [database name];
Создать базу данных:

CREATE DATABASE [database name];
Показать всех пользователей:

SELECT User,Host FROM mysql.user;
Переменовать пользователя magento в puravita2:

RENAME USER 'magento'@'%' to 'puravita2'@'%';
